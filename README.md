### 💙간단 설명💙
1. SplashView: 0.5초 슉슉
2. EnrollView: 명함 제작용 정보 입력 뷰 (필수 정보 + 개별 정보)
3. FrontView: 필수 정보 포함, 명함 앞면
4. BackView: 스와이프로 선택 정보 포함, 명함 뒷면
5. MatchView: 다른 친구와의 궁합 맞추기
